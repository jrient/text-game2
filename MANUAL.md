## 游戏简介

《PixelSurvivor》是一款生存割草类游戏。你将控制一名像素风格的角色，在充满敌人的战场上生存下去。通过击败敌人获取经验值，升级后选择新的武器或技能，不断强化自己，最终击败BOSS或在无尽模式中挑战极限。

## 基础操作

### 键盘控制
| 按键 | 功能 |
|------|------|
| `W` / `↑` | 向上移动 |
| `S` / `↓` | 向下移动 |
| `A` / `←` | 向左移动 |
| `D` / `→` | 向右移动 |
| `ESC` | 暂停/继续游戏 |

### 触屏控制
| 操作 | 功能 |
|------|------|
| 虚拟摇杆 | 控制角色移动方向 |
| 右上角⏸按钮 | 暂停游戏 |

> **提示**：可在主菜单设置中调整虚拟摇杆位置（左侧/右侧）

## 武器系统

游戏中有6种武器，每种都可以通过升级获得额外属性：

### 1. 魔法弹 (Magic Bolt)
- **类型**：主动攻击
- **特点**：向最近敌人发射魔法弹
- **升级效果**：增加发射数量、伤害、射速

### 2. 飞刀 (Throwing Knife)
- **类型**：主动攻击
- **特点**：向移动方向高速飞出飞刀
- **升级效果**：增加飞刀数量、穿透力、伤害

### 3. 火球 (Fireball)
- **类型**：主动攻击
- **特点**：发射缓慢但伤害高的火球，命中时爆炸
- **升级效果**：增加爆炸范围、伤害、弹射次数

### 4. 闪电 (Lightning)
- **类型**：链式攻击
- **特点**：在多个敌人间跳跃的闪电
- **升级效果**：增加跳跃次数、伤害、范围

### 5. 大蒜光环 (Garlic Aura)
- **类型**：被动光环
- **特点**：在角色周围形成伤害区域
- **升级效果**：扩大范围、增加伤害、附加击退

### 6. 旋转斧 (Rotating Axe)
- **类型**：环绕攻击
- **特点**：斧头绕角色旋转，持续造成伤害
- **升级效果**：增加斧头数量、旋转速度、伤害

## 技能与升级

### 升级机制
- 击杀敌人掉落经验球
- 收集经验球获得经验值
- 经验值满后升级，暂停游戏并显示三选一选项

### 技能类型

#### 主动技能
- 直接获取新武器
- 提升现有武器等级

#### 被动技能
| 技能 | 效果 |
|------|------|
| 强健体魄 | 增加最大生命值 |
| 迅捷步伐 | 增加移动速度 |
| 力量强化 | 增加武器伤害 |
| 坚韧护甲 | 减少受到的伤害 |
| 磁力吸引 | 增加经验球拾取范围 |
| 冷却缩减 | 减少武器攻击间隔 |
| 吸血打击 | 击杀敌人回复生命值 |

## 游戏模式

### 关卡模式
- 5个预设关卡，每个关卡有独特的主题和敌人组合
- 限时生存，击败BOSS即可通关
- 适合新手熟悉游戏机制

### 无尽模式
- 无时间限制，敌人强度随时间递增
- 每5波出现精英敌人
- 每10波出现BOSS
- 挑战生存极限

## 技巧与策略

### 新手技巧

1. **优先拾取经验** - 生存的基础是不断升级
2. **多样化武器** - 不同武器互补，覆盖不同攻击范围
3. **保持移动** - 站桩容易被包围
4. **利用无敌帧** - 受伤后有短暂无敌时间

### 进阶策略

#### 武器搭配推荐
- **近战流**：大蒜光环 + 旋转斧 + 闪电
- **远程流**：魔法弹 + 火球 + 飞刀
- **全能流**：魔法弹 + 大蒜光环 + 闪电

#### 升级优先级
1. 早期：优先获取第2-3种武器
2. 中期：提升主武器等级，搭配被动技能
3. 后期：强化输出，补充生存能力

#### 应对BOSS
- 保持移动，避免BOSS直线冲撞
- BOSS半血后会进入狂暴状态
- 优先清理小怪，避免被包围

## 常见问题

**Q: 为什么没有声音？**
A: 请检查主菜单设置中音效是否开启。部分浏览器需要用户交互后才能播放音频。

**Q: 怎么回血？**
A: 升级时选择"吸血打击"被动技能，击杀敌人会回复生命值。

**Q: 无尽模式能玩多久？**
A: 理论上没有上限，但敌人会越来越强。试试你能坚持多少波！

**Q: 数据会保存吗？**
A: 当前版本不保存进度，每次重新开始。设置（音效、摇杆位置）会保存。
